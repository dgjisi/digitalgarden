# 关于定位

### 怎么看定位框与检测框之前的关系？

    解答：在右侧ROI页面中随便点开一个测试的ROI，确认下面的X定位与Y定位
    如果X定位是“-1”，表示当前框水平方向没有定位
    如果Y定位是“-1”，表示当前框坚直方向没有定位
    如果X与Y都是“-1”，则表示当前框在水平与坚直方向都没有定位

如下图，左侧参数中显示X定位“0”，Y定位“1”，表示当前16号框，以0号框数据作水平方向跟踪，以1号框数据作坚直方向跟踪
