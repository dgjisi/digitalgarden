倍数计算公式：
`倍数 = 显示器尺寸 x 25.4 / 6 x 0.5 x (0.7--4.5)`

参数：
1. 显示器尺寸，如上图为15.6寸，而15.6 x 25.4表示将英制单位转换成公制
2. /6 表示相机对角
3. x0.5 表示镜头目镜(镜头上可以看到，通常分0.5与0.35两种)
4. 0.7--4.5表示物镜，取当前所在刻度值

以上倍数仅供参考！
