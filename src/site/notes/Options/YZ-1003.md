# 方案名称：排针检测
配制：
- 相机： 高清600万分辨率    X3
- 镜头： 百万高清镜头    X3
- 光源： LED冷光源    X4
- 主机： 英特尔多接口CCD专用主机（可带多相机）   
- 显示器： 21.5寸高清显示器
- CCD软件： 自主研发
