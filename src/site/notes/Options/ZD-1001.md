# 方案名称：压铸厚度及正位度检测（四相机）
配制：
- 相机： 高清600万分辨率    X6
- 镜头： CY50mm高清镜头    X6
- 光源： LED冷光源 T6020B    X6
- 主机： 英特尔多接口CCD专用主机（可带多相机）   
- 显示器： 21.5寸高清显示器
- CCD软件： 自主研发
